# Учим учиться (Telegram-бот)

## Как запустить?
 - открыть репозиторий
 - создать .env
 - запустить docker-compose
#### Пример как должен выглядеть.env
```
TOKEN="токен"
PG_DATABASE="db"
PG_USER="user"
PG_PASSWORD="password"
ADMINS="admin1_username admin2_username"
```



## Функционал бота

#### Старт
- /start
#### Добавить пост
- /add_post <номер модуля> | <номер темы> | <заголовок> | <содержимое>
#### Добавить к посту картинку, можно несколько
- /add_image <номер модуля> | <номер темы> | <URL картинки>
#### Добавить вопрос к посту
- /add_question <номер модуля> | <вопрос>
#### Информация, которую мы храним о текущем пользователе
- /data
