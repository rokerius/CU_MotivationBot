# CU LearnToLearn Bot

Telegram бот для мотивации и обучения студентов, предоставляющий структурированный контент по различным модулям и темам с интерактивными элементами.

## 🚀 Возможности

- **Модульная система обучения** - 7 модулей с темами для изучения
- **Интерактивная навигация** - удобные кнопки для перехода между темами
- **Система писем** - пользователи могут писать и сохранять письма себе
- **Система вопросов и ответов** - возможность задавать вопросы по модулям
- **Система квизов** - интерактивные тесты с множественным выбором
- **Мультимедийный контент** - поддержка изображений к постам
- **Отслеживание прогресса** - система отмечает пройденные модули и квизы
- **Административные функции** - добавление контента через команды бота
- **Интеграция с Google Sheets** - синхронизация данных с внешними таблицами
- **Расширенная статистика** - детальная аналитика использования бота

## 🚀 Установка и запуск

### Docker развертывание

1. **Настройте переменные окружения:**
   Создайте файл `.env`:
   ```env
   # Telegram Bot
   TOKEN=your_telegram_bot_token
   ADMINS="admin_username1 admin_username2 ..." # никнеймы админов в ТГ, которым доступна admin-панель
   ADMIN_IDS="123456789 987654321 ..." # id админов которым будут рассылаться пользовательские жалобы
   
   # Database
   DB_NAME=your_db_name
   DB_USER=postgres
   DB_PASSWORD=your_password
   
   # Google Sheets Integration
   PRIVATE_KEY=your_private_key
   PRIVATE_KEY_ID=your_private_key_id
   CLIENT_EMAIL=your_client_email
   CLIENT_ID=your_client_id
   AUTH_URI=https://accounts.google.com/o/oauth2/auth
   TOKEN_URI=https://oauth2.googleapis.com/token
   AUTH_PROVIDER_X509_CERT_URL=https://www.googleapis.com/oauth2/v1/certs
   CLIENT_X509_CERT_URL=your_cert_url
   UNIVERSE_DOMAIN=googleapis.com
   PROJECT_ID=your_project_id
   GOOGLE_SHEET_NAME=your_sheet_name
   ```

2. **Запустите с помощью Docker Compose:**
   ```bash
   docker-compose up -d
   ```

## 📊 Структура базы данных

### Таблицы:

- **users** - информация о пользователях и их прогрессе
- **posts** - контент постов по модулям и темам
- **post_images** - изображения к постам
- **module_questions** - вопросы по модулям
- **quizzes** - квизы с вариантами ответов

## 🤖 Команды бота

### Основные команды
- `/start` - запуск бота
- `/reset_my_data` - сбросить информацию о пользователе

### Административные команды
- `/admin` - панель администратора
- `/set_post <модуль> | <тема> | <заголовок> | <содержимое>` - добавить пост
- `/set_image <модуль> | <тема> | <URL>` - добавить изображение к посту
- `/set_question <модуль> | <вопрос>` - добавить вопрос к модулю
- `/update_data` - синхронизация с Google Sheets
- `/get_stat` - статистика использования бота
- `/call_database` - выполнение SQL запросов
- `/update_database` - обновление данных в БД

## 🔧 Конфигурация

### Переменные окружения:
- `TOKEN` - токен Telegram бота
- `DB_HOST` - хост базы данных
- `DB_PORT` - порт базы данных
- `DB_NAME` - имя базы данных
- `DB_USER` - пользователь базы данных
- `DB_PASSWORD` - пароль базы данных
- `ADMINS` - никнеймы администраторов (через пробел) для доступа к админ панели 
- `ADMIN_IDS` - ID администраторов (через пробел) которым будут рассылаться репорты ошибок от пользователей 
- `GOOGLE_SHEET_NAME` - имя Google таблицы для синхронизации

## 🔄 Синхронизация данных

Бот поддерживает синхронизацию с Google Sheets для:
- Постов и их содержимого
- Изображений к постам
- Вопросов по модулям
- Квизов с вариантами ответов

Для настройки синхронизации необходимо:
1. Создать Google Cloud Project
2. Включить Google Sheets API
3. Создать сервисный аккаунт
4. Настроить переменные окружения для Google API
5. Предоставить доступ к Google таблице сервисному аккаунту
